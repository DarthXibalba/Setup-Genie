# Extensions
In VS Code, hit `Ctrl+P` and run the following command:
```
ext install @id:ms-vscode.cpptools-extension-pack
@id:waderyan.gitblame
@id:donjayamanne.git-extension-pack
@id:golang.Go
@id:ms-python.python
@id:timonwong.shellcheck
@id:bierner.github-markdown-preview
@id:bierner.markdown-preview-github-styles
@id:mechatroner.rainbow-csv
@id:mathematic.vscode-pdf
```

This will take you to the extension tab. Hit `Install` on the following packages:
- C/C++ Extension Pack
  - Includes `C/C++` & `CMake Tools` extensions
- Git Blame
- Git Extension Pack
  - Includes `Git History` extension
- Go
- Python
  - Includes `Pylance` extension
- Shellcheck
- Markdown Viewer
- Rainbow CSV
- PDF Viewer
